# notmyidea

This Theme base on Pelican's default theme

### Demo

https://blog.yasking.org

### Changelog

- 优化 Archives 页展示
- 优化 Tags 页的展示
- 添加 Social 展示的 E-mail 图标
- 添加多说, 畅言评论支持


### 配置

开启多说评论

```conf
DUOSHUO_SHORT_NAME = 'short_name'
```

开启畅言评论

```conf
CHANGYAN_APPID = 'appid'
CHANGYAN_CONF = 'conf'
```
